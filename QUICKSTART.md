# 🚀 Hướng Dẫn Bắt Đầu Nhanh

## Chạy Ứng Dụng

### 1️⃣ Cài Đặt Dependencies
```powershell
npm install
```

### 2️⃣ Chạy Development Server
```powershell
npm run dev
```

### 3️⃣ Mở Trình Duyệt
```
http://localhost:5174
```

## 🎯 Các Trang Chính

| Trang | URL | Mô Tả |
|-------|-----|-------|
| Dashboard | `/admin` | Bảng điều khiển tổng hợp |
| Sông | `/admin/rivers` | Quản lý các dòng sông |
| Trạm | `/admin/stations` | Quản lý trạm đo lường |
| Hub | `/admin/hubs` | Quản lý các trung tâm hub |
| Cảm Biến | `/admin/sensors` | Quản lý cảm biến |
| Tham Số | `/admin/parameters` | Quản lý tham số quan trắc |
| Quan Trắc | `/admin/observations` | Xem lịch sử quan trắc |
| Gói Dữ Liệu | `/admin/data-packages` | Quản lý gói dữ liệu |

## 💡 Tính Năng Chính

✅ **Quản lý dữ liệu**: Thêm, chỉnh sửa, xóa dữ liệu  
✅ **Thống kê**: Xem các số liệu thống kê chính  
✅ **Responsive**: Hoạt động trên tất cả các thiết bị  
✅ **Modern UI**: Giao diện đẹp với Ant Design  

## 🛠️ Các Lệnh Khác

```powershell
# Build production
npm run build

# Lint code
npm run lint

npm run preview
```
